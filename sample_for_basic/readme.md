## MMLコンパイル

bgm02.bat がサンプルになります。
コマンドは

### BSAVE形式（アドレスは000hから）
````mc.exe bgm02.txt bgm02.asm bgm02.bin bsave````

* 最後のbsaveはとにかくなんでもよいので文字が入っていればOKです。

### 生バイナリ形式（ヘッダなし）
````mc.exe bgm02.txt bgm02.asm bgm02.bin````

* 4番目のパラメータに何もなければ生バイナリデータでセーブされます。

### バイナリなし。asmのみ
````mc.exe bgm02.txt bgm02.asm````
